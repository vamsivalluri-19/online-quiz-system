# online-quiz-system

## Deploy on Vercel

This project is a static HTML/CSS/JavaScript site, so it can be deployed to Vercel directly from the GitHub repository.

1. Push this repository to GitHub.
2. In Vercel, choose **Add New Project**.
3. Import the GitHub repository `vamsivalluri-19/online-quiz-system`.
4. Keep the default settings and deploy.

The app entry point is `index.html`, and the included `vercel.json` ensures the root URL serves that page.
